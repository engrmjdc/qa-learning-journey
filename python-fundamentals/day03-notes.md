*What is an operator?*
- symbol used to perform operations on variables or values

*Examples of an operator:*
- ( + - * / % = )

*Difference between = and ==*
= : assigning of value
== : used for comparison

*What is a comparison operator?*
- Used to compare two values or expressions and returns True or False.
ex. == , != , > , < , >= , <=

*What is a logical operator?*
- Used to combine or evaluate boolean expressions and returns True or False.
ex. and , or , not
